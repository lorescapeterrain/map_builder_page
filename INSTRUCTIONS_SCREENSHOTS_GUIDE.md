## Screenshot Best Practices

### Settings
- Browser: Chrome (latest)
- Window size: 1920Ã—1080 for desktop
- Scale: 100%
- Hide browser UI (F11)
- Clear background

### Content
- Use a realistic map (15Ã—12 tiles)
- Mix of biomes (Grassland, Arctic, Desert)
- 3 layers minimum
- Varied tile rotations

### Annotations
Use callout boxes with arrows for:
- New features
- Important buttons
- Keyboard shortcuts
- State changes

### Colors for Annotations
- Red: Important actions
- Blue: Information
- Green: Success states
- Yellow: Tips/warnings

## Animated GIFs

### Tools
- ScreenToGif (Windows)
- LICEcap (Mac/Windows)
- Peek (Linux)

### Settings
- FPS: 15-20
- Duration: 3-5 seconds
- Loop: Infinite
- Optimize: Yes
- Max size: 5MB

### Timing
1. Show initial state (1s)
2. Perform action (1-2s)
3. Show result (1s)
4. Pause before loop (0.5s)
